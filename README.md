# PDF to Image Converter

A powerful client-side PDF to Image converter built with React.

## Features

- **Client-side Processing**: All conversions happen in your browser. No files are uploaded to any server.
- **Batch Selection**: Select multiple pages to download or stitch.
- **ZIP Download**: Download selected pages as a single ZIP file.
- **Long Image Stitching**: Combine multiple PDF pages into a single vertical long image.
- **High Quality**: Renders PDFs at 2.0x scale for crisp output.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- PDF.js
- JSZip
- Lucide React Icons

## Setup

This project uses a CDN-based setup for immediate preview but includes a `package.json` for standard development.

To run locally with a bundler:
1. `npm install`
2. `npm start`
